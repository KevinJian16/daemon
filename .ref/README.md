## 当前文档

| 文档 | 用途 |
|---|---|
| **`daemon_实施方案.md`** | 唯一实施规范。所有代码实现以此为准 |
| **`DESIGN_QA.md`** | 设计决策记录。实施方案的设计依据。两者冲突时以 QA 为准 |

| **`DOMINION_WRIT_DEED.md`** | Dominion-Writ-Deed 机制的唯一权威文档 |
